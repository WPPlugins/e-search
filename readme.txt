Extended Search functionalify for WordPress

== Description ==

This E-Search plugin may filter your WordPress content by several fields at the same time (category, tag, text, date, author, title a-z).

The plugin demo can be found at [http://wpcar.net](http://wpcar.net).

== Installation ==

1. Download the Plugin to your desktop.
2. If downloaded as a zip archive, extract the Plugin folder to your desktop.
3. Read through the "readme" file thoroughly to insure you follow the installation instructions.
4. With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
5. Go to Plugins screen and find the newly uploaded Plugin in the list.
6. Click Activate Plugin to activate it.
7. Go to Widgets screen and activate the Plugin widget.
